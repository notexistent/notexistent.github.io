Notexistent statement

There are some very important rules. This rules aimed at making unified network of artwork. Only pages matches this rules can join this network. It helps create reliable frame of algorithmic hardness that allow partisipants lean on it by creating and publishing they works.

For adding the artwork you should create a page, check this page match this rules. In particular, page must include link to this document or document itself, page must link to other artwork page. After that you must notify other participants of notexistent to make a link to your artwork.

At any time if any work that have no at least one link to itself exist more than 24 hours all other works can no longer be part of notexistent. It means that all participants of notexistent should support connectedness of artwork.

There is no way to change this document durning Notexistent. There is no organizing committee or single curator at notexistent. Only information and rules containment in this document control evolution of Notexistent. All control processes of notexistent occur by non-hierarchical interaction between partisipants.

Notexistent Rules. You can read original text in raw form

  1. Artwork Page is part of Notexistent only if it corresponds to this rules
  2. Artwork must be available without any offline interaction with allocated place or peoples
  3. If Artwork is not a webpage itself, it must be associated with its webpage: page must contain description of Artwork and way to access to it, in Artwork must be information about its page
  4. Participant must express his affiliation to Notexistent by including link to this document or document itself in artwork page in form <a href="%link%" rel="notexistentmanifest">...</a>
  5. Artwork should have a page that one can create link to
  6. In his works Authors must not violate the laws of the countries in which they are citizens
  7. There are no other limitations of format and content
  8. Notexistent is held between %start date% and %end date%. Participant must provide availability of his artwork durning notexistent
  9. The author must indicate his contact email in page code in form <meta name="nonexistentauthor" content="%email%">
  10. Artwork page must have at least one link to others artwork with explicit indication that link target is Artwork of Notexistent by adding "[Notexistent]" to link text and attribute rel="notexistent" to link tag. Must not be any link marked this way if link target is not Artwork of Notexistent. If your artwork have link in specific non-html form (javascript for example), you must duplicate html link in your page code to allow automatic check connectedness of Notexistent
  11. Must be possible to get (directly or indirectly) to any other Artwork of notexistent from Artwork page. For new Artwork, there may be no links within 24 hours
